# Очередь тем для блога Buy_USDTs

Правила авто-публикации (1 статья в день, берём первую тему из «В очереди»):
- Пишем по `_template.html` и системному промпту в `scripts/generate-article.mjs`
  (он уже включает копирайтинг- и SEO/GEO-скилы: см. `scripts/copywriting-skill.md`, `scripts/seo-geo-skill.md`).
- Стандарт статьи: **answer-first** (прямой ответ в первом абзаце), H2-вопросы, списки/таблица,
  закрытие возражений ВЭД, раздел «Частые вопросы» (FAQ-разметка ставится автоматически), один ценностный CTA. Объём 800–1200 слов.
- ⚠️ YMYL: без юридических/финансовых гарантий и обещаний доходности; юр./налоговые моменты — аккуратно, «обзорно».
- Заголовки — по формулам: выгода + закрытие страха + конкретика/число.
- Опубликованную тему скрипт сам переносит в «Опубликовано».
